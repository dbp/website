3.2 Homework 2

3.2.1 Problem 1

Consider a data definition, TrafficLight, as follows:

; A TrafficLight is one of:
; - 'green
; - 'yellow
; - 'red
; Interpretation: Color of light that is illuminated
(define TL-G 'green)
(define TL-Y 'yellow)
(define TL-R 'red)
 
#;(define (tl-temp tl)
  (... (cond [(symbol=? tl TL-G) ...]
             [(symbol=? tl TL-Y) ...]
             [(symbol=? tl TL-R) ...])))

Your task is to write a specification traffic-light-step-spec that describes formally the behavior of a function traffic-light-step, which has signature TrafficLight -> TrafficLight and changes the lights from green to yellow to red to green.

Once you write the spec, implement the function traffic-light-step, and prove using check-expect that your code matches your specification.
